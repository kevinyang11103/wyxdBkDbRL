# 仓库管理系统 python176

## 项目概述

本项目是一款基于Python Django和Vue3的仓库管理系统，旨在提供便捷的商品管理、商品分类管理和用户管理等功能。

## 技术栈

- 后端：Django
- 前端：Vue3、Ant-design-vue
- 开发工具：Pycharm
- 后端构建工具：Pip
- 前端构建工具：vite

## 运行环境

- 操作系统：Windows
- Python版本：3.8
- NodeJs版本：v16
- 数据库：mysql8.0

## 功能模块

- 商品管理
- 商品分类管理
- 用户管理

## 系统角色

本项目主要面向以下角色：

- 管理员：负责商品、商品分类和用户的管理
- 普通用户：可以查看商品信息

## 目录结构

```plaintext
项目根目录/
├── 后端代码/
│   ├── apps/
│   │   ├── 商品管理/
│   │   ├── 商品分类管理/
│   │   └── 用户管理/
│   └── ...
└── 前端代码/
    ├── public/
    ├── src/
    │   ├── assets/
    │   ├── components/
    │   ├── views/
    │   └── ...
    └── ...
```

## 核心亮点

- 集成Ant-design-vue UI库，界面美观、易用
- 支持商品、商品分类和用户管理的功能，满足仓库管理的日常需求

## 部署步骤

1. 准备运行环境，安装Python 3.8、NodeJs v16和mysql8.0
2. 克隆项目代码到本地
3. 进入后端代码目录，使用Pip安装依赖
4. 使用Pycharm运行后端项目
5. 进入前端代码目录，使用vite构建项目
6. 启动前端项目，访问指定地址即可进入系统
